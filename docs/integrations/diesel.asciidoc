== Annotations

Use `#diesel(...)` annotations to control code generation:

[source,immigrant]
----
#diesel(type="dt::Text",copy,custom)
scalar user_id = sql"TEXT";

#diesel(derive = "async_graphql::Enum")
enum client_kind { confidential; public; };
----
